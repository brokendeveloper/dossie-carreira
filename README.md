# Dossiê de Carreira — Luccas Fernandes

Site estático de portfólio e dossiê de carreira, desenvolvido como parte da disciplina **Projeto 5** do curso de Análise e Desenvolvimento de Sistemas na [CESAR School](https://www.cesar.school).

## Estrutura

```
dossie-carreira/
├── dossie.html          # Documento principal
├── assets/
│   ├── css/styles.css   # Estilos
│   └── js/main.js       # Interações (fade-up, scroll dots, pills)
└── README.md
```

## Como abrir

Abra `dossie.html` diretamente no navegador — sem dependências, sem build.

## Seções

| # | Seção | Conteúdo |
|---|-------|---------|
| 01 | Perfil Profissional | Formação, skills técnicas, projetos |
| 02 | Diagnóstico | Autodiagnóstico e SWOT pessoal |
| 03 | Contexto e Mercado | Pesquisa de vagas reais e gap analysis |
| 04 | Hipóteses | Objetivos por horizonte e plano de ação |
| 05 | Experimentação Prática | Projeto Integrador e soft skills evidenciadas |
| 06 | Evidências | Certificados, declarações e feedbacks |
| 07 | Percurso Acadêmico | Linha do tempo semestral |
| 08 | Pós-Faculdade | Caminhos e plano concreto pós-formatura |
| 09 | Síntese Final | Comparativo de evolução e carta profissional |
